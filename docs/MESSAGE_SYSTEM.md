# Message system — WDIWF / Jackye Clayton

**Status:** Active working guide (not background reading).

**Precedence when anything conflicts:** **MESSAGE → VOICE → DESIGN**

---

## Rule 1: Message before design

No layout, visual gimmick, or format is chosen until the message is locked.

Before writing or briefing an image model, answer in plain language:

1. **What truth** is this piece telling? (One sentence. No slogan.)  
2. **Why it matters** to a worker, job seeker, or people leader *this week*.  
3. **What people are missing** — the gap, myth, or asymmetry the piece corrects.  
4. **Who we mirror** — which reader situations does this speak to *by name* (role, sector, power dynamic, risk)? Not “everyone.”  
5. **Values / mission fit** — how does this advance WDIWF’s mission and **radical inclusion** (`newsletter/WDIWF_NEWS_CHARTER.md`)? If it doesn’t, why are we shipping it?  
6. **Customization** — what **known signals** (employer, industry, user segment, geography) can we use honestly? If none, write for a clear **public** mirror, not fake “you” personalization.

If the message could apply to any brand or any week, **stop** and rewrite until it is **specific**. If it doesn’t **mirror** anyone’s real corner of work, deepen the framing before design.

---

## Rule 2: Then choose format

Only after the three answers are solid:

| If the job is to… | Prefer |
|-------------------|--------|
| Correct a factual / narrative gap in the news cycle | Copy-led brief + source map (see `newsletter/WDIWF_NEWS_CHARTER.md`) |
| Stop the scroll with one stake | Short social / LinkedIn tickle (`newsletter/template-linkedin-tickle.md`) |
| Earn share + “omg Jackye” with visual punch | Poster path: `VIRAL_POSTER_IMAGE_SPEC.md` + `visual-references/` |
| Deepen trust over time | Substack / site longform; Friday mail (`CHANNEL_PLAYBOOK.md`) |

Format serves message — never the reverse.

## Gaps / escalate

If the truth is unclear, sources are thin, or the ask conflicts with charter or inclusivity, **flag early** — do not pad with generic “future of work” language.
