# WDIWF BUILD OS

## PRODUCT TRUTH
WDIWF is a trust product.

It helps people understand who they are really working for using public records.

It is not:
- a review site
- a content platform
- an opinion engine

Everything must:
- support a real decision
- show evidence
- keep uncertainty visible
